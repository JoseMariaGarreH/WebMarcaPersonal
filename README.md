# Proyecto React

Este proyecto está desarrollado con React.

## Instalación

Para instalar las dependencias necesarias, ejecuta los siguientes comandos:

```sh
npm install
npm install react-router-dom
npm install react-hook-form
```

## Ejecución

Después de instalar las dependencias, puedes iniciar el proyecto con:

```sh
npm run dev
```

## Requisitos

- Node.js instalado en tu sistema.
- npm instalado (incluido con Node.js).
